# 📺 ESP32 + ST7735 TFT (SPI)
### Beginner to Expert Display Programming

---

## 📌 Overview
This project demonstrates interfacing the **ST7735 TFT display** with an **ESP32** using the **SPI protocol**.

The repository follows a structured learning path:
- RGB color manipulation
- Image display
- Simple image processing
- Live image streaming and processing

Originally developed for **Mini Project 2025** at **SPIT Mumbai**.

---

## 🎯 Objectives
- Interface ST7735 TFT with ESP32 using SPI
- Understand pixel-level RGB control
- Render images on TFT
- Perform basic image processing
- Implement live image streaming

---

## 🛠 Hardware Requirements
- ESP32
- ST7735 TFT Display (SPI)
- Jumper wires
- Power supply

---

## 💻 Software Requirements
- Arduino IDE 
- Required TFT libraries
- Serial Monitor
